#  Full Stack Task Manager (React + Spring Boot)

A complete full stack Task Management application with **React TypeScript frontend** and **Spring Boot backend**.
Supports authentication with **JWT + Refresh Tokens**, task CRUD, drag-drop status updates, and real-time updates using **WebSocket**.

##  Features
- JWT Authentication + Refresh Tokens
- Protected Routes (Frontend)
- Task CRUD Operations
- Drag & Drop Kanban Board
- Real-time updates via WebSocket
- Responsive UI (Tailwind / MUI)
- Dockerized Full Stack
- Testing + CI Pipeline

##  Tech Stack
### Backend
- Java 17, Spring Boot 3.x
- Spring Security + JWT
- Spring Data JPA
- PostgreSQL
- WebSocket

### Frontend
- React 18 + TypeScript
- Vite
- Axios
- Zustand/Context
- Tailwind CSS / MUI
- Drag & Drop
